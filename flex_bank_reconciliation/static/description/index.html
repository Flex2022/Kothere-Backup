Features

Packed with all the features. Tailored for Odoo 16.

Transaction Matching: Match bank statement lines with existing transactions in Odoo based on amount, reference, and date. Propose potential matches for manual confirmation. Allow users to manually match bank statement lines with Odoo transactions.
Provide a user-friendly interface for manual matching with search and filter options. Bank Statement Line Creation: Ability to create new transactions directly from unmatched bank statement lines. Option to split a single bank statement line into multiple Odoo transactions.
Reconciliation Reports: Generate detailed reconciliation reports showing matched, unmatched, and potentially erroneous transactions. Provide summary reports for management review.
Reconciliation Dashboard: A dashboard view showing the reconciliation status, including the number of matched, unmatched, and problematic transactions. Graphical representation of reconciliation progress over time. Multi-Currency Support.
Handle bank accounts in different currencies. Automatic currency conversion based on daily rates. Bank Fees & Interest Handling.
Ability to automatically recognize and categorize bank fees, interest income, and other bank-initiated transactions. Reconciliation Templates.
Create templates for recurring or common reconciliation scenarios to speed up the process. Audit Trail: Maintain a detailed log of all reconciliation actions, including who performed them and when. Provide a rollback or undo feature for erroneous reconciliations.
Integration with Other Odoo Modules: Seamless integration with the accounting module for transaction posting. Integration with the contacts module for payee and payer details.
Notifications & Alerts: Notify users of new bank statements available for reconciliation. Alert users of potential reconciliation issues or discrepancies.
Permissions & User Roles: Define user roles for different reconciliation tasks, such as importing bank statements, matching transactions, and generating reports. Ensure sensitive financial data is accessible only to authorized personnel.
Bank Account Management: Manage multiple bank accounts and their details. Set up account-specific reconciliation rules and preferences.
Periodic Reconciliation: Set up periodic (e.g., monthly) reconciliation schedules. Lock reconciled periods to prevent changes.
